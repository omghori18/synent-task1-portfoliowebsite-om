## ✨ Features

- **Premium Modern Design:** Deep sleek dark mode palette (`#121418` theme) using modern typography (Inter and Raleway) from Google Fonts.
- **Dynamic CSS Parallax Orbs:** Multiple fixed gradient orbs that slide smoothly at different rates in response to scroll events, creating a rich 3D depth effect.
- **Pure CSS Mobile Navigation:** Sidebar drawer toggled via a hidden checkbox hack, keeping the JavaScript footprint minimal.
- **SEO & Social Optimization:** Fully configured with metadata, keywords, description tags, and Open Graph titles for social media sharing.
- **Responsive Layout:** Responsive flex and grid systems designed for standard screens, tablets, and mobile devices.
- **Micro-interactions:** Smooth transitions, hover scaling, and interactive buttons for higher user engagement.

## 💻 How to Run Locally

Since this is a static web application, you do not need complex build systems. 

### Method 1: Direct File Open
You can open the [index.html](file:///Users/ghoriommukeshbai/Desktop/internship/index.html) file directly in any modern web browser.

### Method 2: Local Static Server (Recommended)
To prevent CORS or path issues if you expand this project, run a local development server using Node.js:

1. Install static server helper (if not installed):
   ```bash
   npm install -g local-server
   ```
   *or* run via npx:
   ```bash
   npx local-server
   ```
2. Or use Python's built-in HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
3. Open [http://localhost:8000](http://localhost:8000) in your browser.
